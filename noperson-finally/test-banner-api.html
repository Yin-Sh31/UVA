轮播图接口测试

问题分析

根据错误日志，您正在调用的路径 /api/banner/list 不存在于系统中。

正确的轮播图接口路径是：/api/banners（注意是复数形式）

此接口需要一个必需的参数 type，用于指定用户类型（farmer=农户 或 flyer=飞手）

接口测试

使用说明

微信小程序调用示例：

// 微信小程序中正确调用轮播图接口的代码示例
wx.request({
  url: 'http://your-backend-server/api/banners',
  method: 'GET',
  data: {
    type: 'farmer' // 或 'flyer'，根据用户类型设置
  },
  success(res) {
    console.log('轮播图数据:', res.data);
    // 处理轮播图数据
    const banners = res.data.data || [];
    // 更新页面数据
    that.setData({
      bannerList: banners
    });
  },
  fail(err) {
    console.error('获取轮播图失败:', err);
  }
});

注意事项：

  • 确保在小程序后台配置了正确的服务器域名白名单
  • 接口不需要登录即可访问（根据代码分析）
  • 返回的图片URL格式为 '/images/xxx.jpg'，需要根据实际部署情况进行处理
  • 如果需要管理员操作轮播图，需要使用对应的管理员接口并携带JWT Token
